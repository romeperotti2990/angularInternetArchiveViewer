This is an internet archive view/use tool. It should be designed to search internet archive and use the files it finds without the user having to download them. For example, if I find a .gba file, we should be able to play it with emulator js. If I find a .mp3 file, I should be able to listen to it in the browser.

INSTRUCTIONS:
Currently this is an empty angular project with one empty component in /app/components/item.
